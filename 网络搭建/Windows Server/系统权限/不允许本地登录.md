# 不允许本地用户

允许该用户作为文件共享的认证用户，但不允许使用该用户登录操作系统。

- 按下`Win + R`键打开“运行”对话框，输入`gpedit.msc`，然后按Enter键打开组策略编辑器。

```
计算机配置 -> Windows 设置 -> 安全设置 -> 本地策略 -> 用户权限分配
```



在右侧找到策略名为“**不允许本地登录”**，将不允许本地登录的用户添加到这里面。
